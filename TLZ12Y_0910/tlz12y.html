Hello World

  • A projektunk taroljat Repositorynak hivjak
  • A verziokovetes a Commit-tal valosul meg
  • Parhuzamos fejlesztési ágak Branchekben valosulnakmeg
  1. git init - uj repo
  2. git clone - tavoli repo klonozasa helyi gepre
  3. git status - aktualis repo allapot megnezese
  4. git add - valtoztatasok hozzadasa a kovetkezo commithoz egy fajl/minden
  5. git commit -m - commit uzenet
  6. git branch - meglevo branchek
  7. git branch - uj ag letrehozas
  8. git checkout - agvaltas
  9. git merge - ag egyesites
  1. git remote add origin - tavoli repo hozzadasa
  2. git push origin main - githubra valtoztatsok feltoltese (alapertelmezett ag feltoltese)
  3. git push -u origin main - elso push, hogy a kapcsola rogzuljon
  4. git pull origin main - tavoli repo frissitese
  5. git fetch --all - tavoli repo agak listazasa
  • GitLens - részletes git történet
  • GitHub Pull Requests and Issues - közvetlenül VS Code-ból lehet PR-t nyitni, issue-t kezelni
  • Git History - commitok vizuális böngészése
  • Remote Repositories - közvetlen GitHub repo megnyitása klónozás nélkül
  • Git Graph - Ag vizualizacio
  • Prettier / ESLint - kód formázás és stílusellenőrzés
